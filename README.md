# 🗂️ Smart Data Organiser
### File-Based Dictionary Processing System in Python

## Overview
A Python-based data processing tool that reads structured data from a file, 
validates and transforms it, and outputs an organised, inverted representation.

## Features
- Reads structured key-value data from a text file
- Validates input and handles file errors gracefully
- Supports multiple values per key
- Inverts dictionary relationships dynamically
- Writes clean output to a file
- Logs processing activity

## Technologies
- Python 3
- File I/O | Exception Handling | Data Structures

## How to Run
1. Clone the repository
2. Add your `dictionary.txt` in the same folder
3. Run: `python smart_data_organizer.py`
